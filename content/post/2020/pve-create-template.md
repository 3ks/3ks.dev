# 模板和虚拟机的区别

虚拟机可以 `不可逆` 的转换为模板，模板无法开机，但可以通过模板克隆出新的虚拟机。

虚拟机和模板都可以克隆虚拟机，区别在于模板可以选择 `链接克隆` 和 `完整克隆`。而虚拟机只能 `完整克隆`。
