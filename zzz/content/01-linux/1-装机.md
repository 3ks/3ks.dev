> SSR脚本

`wget -N --no-check-certificate https://raw.githubusercontent.com/ToyoDAdoubiBackup/doubi/master/ssrmu.sh && chmod +x ssrmu.sh && bash ssrmu.sh`

# 配置 HTTP/S FTP 代理

1.  vim /etc/profile

> 文件末尾追加内容 


```
https_proxy=192.168.190.2:1080
http_proxy=192.168.190.2:1080
ftp_proxy=192.168.190.2:1080
export https_proxy
export http_proxy
export ftp_proxy
```

2.  source /etc/profile

> 立即生效，不需要代理时需要注释或者删除配置代码，重复步骤2


# 安装 SSR 客户端

### 1. 下载 SSR 客户端
```git clone https://github.com/ssrbackup/shadowsocksr```

> 需要先安装 `Git`

### 2. 创建填写配置文件
vim /etc/shadowsocks.json
```
{
    "server": "68.183.168.80",
    "server_ipv6": "::",
    "server_port": 2333,
    "local_address": "127.0.0.1",
    "local_port": 1080,
    
    "password": "asldkfjlKJHLLJ",
    "method": "aes-256-cfb",
    "protocol": "auth_aes128_md5",
    "protocol_param": "",
    "obfs": "plain",
    "obfs_param": "",
    "speed_limit_per_con": 0,
    "speed_limit_per_user": 0,

    "additional_ports" : {}, // only works under multi-user mode
    "additional_ports_only" : false, // only works under multi-user mode
    "timeout": 120,
    "udp_timeout": 60,
    "dns_ipv6": false,
    "connect_verbose_info": 0,
    "redirect": "",
    "fast_open": false
}
```

### 3.运行客户端 
`nohup sslocal -c /etc/shadowsocks.json /dev/null 2>&1 &`
