## 为什么是 3ks

我的 GitHub ID 是 [3ks](https://github/3ks)，所以就申请了 [3ks.dev](https://3ks.dev) 域名

为什么是 3ks 呢？很简单，因为短，对于 ID 这种东西，个人认为越短越好（深受 QQ 号荼毒）

并且 3ks 的中文发音与 tks 相同，意为`谢谢`。同时，`谢谢` 也是我的口头禅

别问我为什么不用 tks，我是不会告诉你已经被别人占用了的😒
